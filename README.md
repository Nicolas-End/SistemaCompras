## 📌 Sobre o Projeto

Sistema desenvolvido para **gestão de compras**, permitindo o controle completo de:

* Criação de orçamentos
* Gestão de fornecedores
* Controle de pedidos
* Acompanhamento de status
* Associação de itens e anexos

O objetivo é centralizar e organizar o fluxo de compras dentro de uma empresa, garantindo **rastreabilidade, consistência e controle**.
